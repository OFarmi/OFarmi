## Hi there 👋

Here I'll share some of my projects, contributions and where I'm looking to progress.

## 🧠 Personal Projects
- [OFarmi/league-items](https://github.com/OFarmi/league-items), Web app built in React that displays up to the top 5 winrate items for all League of Legends champions.
  - Gave me hands-on experience with understanding REST API and basic React.

## 🤝 Contributions
- [OFarmi/minesweeper](https://github.com/OFarmi/minesweeper), Minesweeper web app capable of storing game states and accounts.
  - Added the capability of rewinding moves after a loss to review errors, account creation and continuing a previous session linked to an account.
  - Gave me experience with Flask and understanding websockets.
- [Capstone-2021-Wristband-Project/firmware](https://github.com/Capstone-2021-Wristband-Project/firmware), Added Bluetooth Low Energy capability to ESP32 processor
  - Taught me how Bluetooth devices advertise their presence via a unique ID, waiting for a possible connection.
  - Using an iOS app, the motor connected to the ESP32 can have its vibration intensity adjusted.
- [neu-cs4530/team-project-group-4i](https://github.com/neu-cs4530/team-project-group-4i), Added status and direct message features to existing web app: covey.town
  - Exposed me to a larger-scale codebase.
  - Taught me the importance of code readability and comments, as large blocks of long-line code can get overwhelming.
- [reversi-team3/pythonConsole](https://github.com/reversi-team3/pythonConsole), Created a desktop app Reversi game capable of running a local and AI mode.
  - Taught me the importance of modular code and how much easier extending features becomes.
  - Experience with TKinter.
